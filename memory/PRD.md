# Armin Shopfitting - Landing Page PRD

## Original Problem Statement
Build a landing page for a construction company called "arminshopfitting". They deal with property analysing, refurbishing, and construction.

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Framer Motion + Shadcn/UI
- **Backend**: FastAPI (Python)
- **Database**: MongoDB
- **Design System**: Industrial/Professional aesthetic with Barlow Condensed + Manrope fonts

## User Personas
1. **Property Developers** - Looking for construction and shopfitting services
2. **Commercial Retail Owners** - Need refurbishing and fit-out services
3. **Real Estate Investors** - Seeking property analysis services

## Core Requirements
1. Professional landing page showcasing construction services
2. Contact form for inquiries
3. Portfolio/Projects showcase
4. Client testimonials
5. Mobile responsive design

## API Endpoints
- `GET /api/projects` - Fetch project portfolio
- `GET /api/testimonials` - Fetch client testimonials
- `POST /api/contact` - Submit contact inquiry
- `GET /api/contact` - List contact inquiries
